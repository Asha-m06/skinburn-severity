# skinburn-severity
AI-based skin burn severity detection system that uses deep learning (CNN) to classify burn images into different severity levels for accurate and early diagnosis.
# Skin Burn Severity Detection using Deep Learning

#description
This project presents a deep learning-based approach for detecting and classifying the severity of skin burns from images. A Convolutional Neural Network (CNN) model is trained to categorize burn images into different severity levels such as first-degree, second-degree, and third-degree burns. The system is intended to support early diagnosis and assist in medical decision-making.

#objective
The objective of this project is to develop an automated system that can accurately classify burn severity using image data, reducing manual assessment time and improving diagnostic efficiency.

#methodology

#results
The model achieved an accuracy of approximately **72%** on the validation dataset. The results indicate that the model is capable of identifying relevant patterns in burn images, while also highlighting opportunities for improvement through techniques such as data augmentation, hyperparameter tuning, and transfer learning.
